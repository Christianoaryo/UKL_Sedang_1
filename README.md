# Program ini berfungsi untuk menghitung faktorial dari sebuah bilangan bulat yang dimasukkan oleh pengguna

1. Scanner berfungsi untuk memasukkan nilai awal
2. Jika bilangan yang dimasukkan negatif, program menampilkan pesan bahwa faktorial tidak dapat dihitung.
3. Jika bilangan nol atau positif, program:
- Menghitung faktorial menggunakan perulangan for.
- Menampilkan proses perkalian satu per satu (misalnya: 1 x 2 x 3 x 4 x 5 =).
- Menampilkan hasil akhir faktorial.
